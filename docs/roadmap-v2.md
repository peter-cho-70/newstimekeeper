## V2 로드맵: 협업/확장(계정/클라우드)

### 목표
- MVP(로컬 파일 기반)에서 검증된 편집/진행 흐름을 유지하면서, 팀 운영에 필요한 **공유/권한/이력**을 제공한다.

---

## 1) MVP 대비 추가 기능
### 1.1 로그인/권한
- 사용자 계정(사번/이메일 기반)
- 권한(예): `admin`, `editor`, `viewer`
- 프로그램/템플릿 편집 권한 분리(템플릿은 제한된 사용자만 편집)

### 1.2 클라우드 저장소
- 템플릿/큐시트 중앙 저장
- 프로그램/날짜/회차별 목록 조회
- 자동 저장(주기적) + 수동 스냅샷

### 1.3 변경 이력/버전
- 편집 히스토리(누가/언제/무엇을)
- 버전 되돌리기(최근 n개)
- 비교(시간 합계, 아이템 수/길이 변화)

### 1.4 실시간 협업(선택)
- 동시 편집(커서/잠금/충돌 해결)
- 최소 구현: “현재 편집자 잠금(1명만 편집)” + 나머지는 읽기

### 1.5 운영 편의
- 템플릿 배포/승인 워크플로(검수 후 배포)
- 프로그램 추가 적용(새 프로그램 생성/복제)

---

## 2) 데이터/기술 고려사항(요구사항 관점)
- 오프라인 대비: 네트워크 끊김 시 로컬 캐시에서 지속 편집 후 재동기화
- 감사/추적: 누가 무엇을 바꿨는지 로그(방송 업무 특성상 중요)
- 보안: 조직 계정 연동(SSO) 가능성(추후)

---

## 3) 릴리즈 단계 제안
- **V2.0**: 로그인 + 클라우드 저장 + 버전 스냅샷(협업의 80% 해결)
- **V2.1**: 권한/승인 워크플로 + 프로그램 확장(템플릿 관리 체계)
- **V2.2**: 실시간 협업(필요 시)
